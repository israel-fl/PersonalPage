AWS Glacier Web App

FLASK-PYTHON-SQLALCHEMY


This is a web interface in the form of a dashboard to facilitate the upload, retrieval and storage of archives uploaded to AWS Glacier.
This app has been highly tailored to be integrated into the Kaseya suite for IT management, and hence, many of the features are meant to be used with the software.

Overall Structure



This was highly customized to represent the structure within the Kaseya suite. You must first start by adding an organization under Configuration > Add Organization. Here you add the Secret Key, Access Key, and Account ID required to interface with AWS services.

Then you add an agent to the organization, including the name of the vault you want to backup the specific machine to, and the location of the backups. More detailed documentation to follow.

Note

This web app has been built in such a way that it interfaces with the other script I wrote called glacier.py

Green Facades (Android App)

Previous Next


Green Facades is meant to be used as an educational tool for those with an interest in green facades. It is meant to provide unified information about both past and current green facades research integrated into a taxonomy to follow design guidelines, strategies, and sustainable education. You can select your parameters and get a research based design recommendation based on your preferences.

The app provides a list of the USDA Zones used for agricultural research. The user then selects a zone that most closely corresponds to their location, or the location in which they wish to grow. The user is then presented with a list of parameters such as plant life expectancy, cost of maintenance, and energy savings possibilities.

Through an integration with the Parse database, the app then searches through the server to retrieve the information of both plants and available systems for vegetation growth.

Although this feature is still in progress, the user will then be able to save the plants they like to a list of their "Favorites".

The app also provides previous case studies performed on different Green Facades systems. The user may also add their own case studies, which are saved locally.

  • Date: Spring 2016
  • Repository: Github
  • Play Store: Green Facades

Soul Voyage

Previous Next



Mobile App Game



Strange Attractor:

One player, four friends - coordinate with your team to guide a ship through the realms of space with the use of magical forces.

Basic Summary:

There will be controls at all four corners of the screen (like a magnet that can be turned on or off), which allows for multiple people to control a single character. There will be obstacles and enemies to avoid/defeat, which may differ depending on the game mode.

Major Features:



Main character is controlled by four buttons on the corners of the screen, which act as magnets that “pull” the characters in that direction.

Particle systems to illustrate what direction the player is being pulled

Procedurally generated obstacles/enemies that make the game harder over time / as you progress

Different types of obstacles and enemies, for example:

Obstacles can block the way so that the player cannot move in that direction

Obstacles can have a force (attractive or repulsive) that affects the character and must be counteracted by the players

Obstacles that change the behavior of the controls (for example, instead of acting like magnets that pull the character in, the controls may switch over to acting like air jets that push the character away)

Obstacles (passive enemies) such as mines that do damage to the character

Enemies (active enemies) that actively try to follow and damage the character

Multiple play modes

Game Center integration for high scores and achievements.

Art Considerations:

The game will have a simplistic and clean art style, featuring mostly monochromatic characters of common shapes and simple particle systems.

  • Date: Spring 2016
  • Repository: Github
  • Play Store: Soul Voyage
